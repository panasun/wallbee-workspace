# Playbook: Onboarding พนักงานใหม่

**เจ้าของ:** HR | **อัปเดต:** 2026-01-01

---

## Day 1
- [ ] รับ equipment (laptop, badge)
- [ ] ตั้ง email และ password ระบบต่างๆ
- [ ] แนะนำทีมและ office tour
- [ ] อ่าน HR Policy, IT Security Policy
- [ ] Start WallBee Bot: https://t.me/wallbee_bot
- [ ] HR แจ้ง IT เพิ่ม telegram_chat_id เข้าระบบ

## Week 1
- [ ] Meeting กับหัวหน้าโดยตรง — รับ KPI และ goals
- [ ] เรียนรู้ระบบ CRM / ERP (ถ้าเกี่ยวข้อง)
- [ ] อ่าน knowledge base ของแผนก
- [ ] Shadow เพื่อนร่วมทีม 3 วัน

## Month 1
- [ ] ส่ง 30-day report ให้หัวหน้า
- [ ] ตั้ง personal OKR และ KPI
- [ ] เข้าร่วม team meeting อย่างเต็มตัว

## Checklist IT
- [ ] สร้าง email account
- [ ] เพิ่มเข้า MS Teams / Line group
- [ ] เพิ่ม Telegram ID เข้า WallBee allowFrom
- [ ] เพิ่มใน USERS.csv
- [ ] สร้าง folder users/{email}/ ใน GitHub

## Checklist HR

- [ ] แบบฟอร์มประกันสังคม
